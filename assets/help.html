MAX POST
Руководство пользователя

Программа для массовой рассылки объявлений в группы мессенджера MAX и ВКонтакте. Поддержка текста, фото, шаблонов и умной рассылки по адресам.

🚀Быстрый старт

При первом запуске программа попросит ввести токены доступа. Без них отправка работать не будет.

Запустите MAX POST.exe. Откроется главное окно программы.
Нажмите кнопку ⚙️ (шестерёнка) внизу боковой панели → откроется диалог настроек.
Введите MAX Token и/или VK User Token. Нажмите ✓ Проверить рядом с каждым.
Вернитесь в главное окно. Введите текст → выберите адреса → загрузите фото → нажмите Опубликовать.
Файл реестра адресов max_address.xlsx находится в папке с программой. Откройте его в Excel и заполните адреса, ссылки и ID групп.

🔑Токены доступа

Откройте ⚙️ Настройки в боковой панели. При первом открытии пароль пустой — нажмите ОК.

Токен MAX

В настройках найдите поле MAX Token.
Нажмите 🔗 Получить токен — откроется страница авторизации в браузере.
Скопируйте токен из браузера, вставьте в поле и нажмите ✓ Проверить.

Токен ВКонтакте

Токен ВК действителен около 3 месяцев. Программа напомнит о смене через уведомление в трее.

После получения нового токена ВК нажмите Действия → 🔑 Сменил токены VK — это сбросит счётчик напоминаний.

Telegram-уведомления опционально

Если указать Telegram Bot Token и Chat ID — программа будет присылать отчёты об отправке прямо в Telegram.

🖥️Интерфейс

Главное окно делится на левую панель (адреса получателей) и правую панель (ввод данных). Граница между ними перетаскивается, размер сохраняется.

Тема оформления

Кнопка 🌙 / ☀️ внизу боковой панели переключает тёмную и светлую тему без перезапуска программы.

✏️Ввод текста

📄
Открытие файла
Кнопка 📄 Файл в заголовке раздела. Поддерживаются .txt (UTF-8, CP1251 — автоматически) и .docx (вставляется чистый текст).
📋
Шаблоны текста
Кнопка 📋 — сохранить текущий текст как шаблон или вставить сохранённый. Подробнее в разделе «Шаблоны».
🔢
Счётчик символов
Отображается под полем ввода.
обычный   близко к лимиту   превышен
📤
Отправить выделенный фрагмент
Выделите любой кусок текста в поле ввода → нажмите ПКМ → выберите «📤 Отправить выделенное в MAX» или «📤 Отправить выделенное в ВКонтакте».

Программа покажет диалог подтверждения с превью фрагмента и отправит только выделенный текст (+ текущее фото, если загружено), не затрагивая основной текст в поле ввода.
Удобно когда один большой текст нужно отправить частями в разные группы: выделите нужный абзац → ПКМ → отправить.

Предпросмотр

Справа от поля ввода — карточка предпросмотра. Показывает как пост будет выглядеть с фото и текстом в реальном времени.

Перетаскивайте границу между левой и правой панелью — ширина сохраняется между запусками программы.

📍Адреса получателей

Список получателей формируется в левой панели. Галочка рядом с адресом — он включён в рассылку.

Поиск по реестру

Введите улицу или номер дома в поле 🔍 Поиск — появится список совпадений с чекбоксами.

Мультивыбор адресов
Введите «есенина» → список совпадений с чекбоксами.
Отметьте нужные адреса галочками.
Нажмите ✓ Добавить (N) — все добавятся сразу.
Если ничего не отмечено — кнопка показывает «Добавить все (N)». Нажмите Enter для быстрого добавления.

Другие способы добавления

✏️
Ручной ввод
Кнопка + Добавить — ввод адреса вручную с автопоиском по реестру.
📋
Вставка списка
Кнопка 📋 в заголовке — вставьте несколько адресов из буфера. Программа разберёт список и покажет диалог выбора.
🔍
Авто-определение из текста
При вводе текста программа автоматически ищет в нём адреса и предлагает добавить. Зелёная подсветка — найден, оранжевая — не найден в реестре.

Выбрать все адреса

Чекбокс Выбрать все адреса в панели рассылки — одним кликом загружает все 192 адреса из реестра в список получателей. Снятие галочки очищает список.

Рекомендуется отправлять не более 10 групп за раз — во избежание временной блокировки в мессенджере MAX (ограничение ~5 минут).

🖼️Фото

📂
Загрузка
Кнопка Загрузить фото находится в левой панели под полем ввода текста. Нажмите или перетащите файл изображения в окно. Поддерживаются JPG, PNG и другие форматы.
📌
Закрепить фото
Кнопка 📌 становится оранжевой когда активна. При очистке формы фото не сбрасывается — удобно для серии рассылок с одним фото.
🗂️
Галерея последних фото
Под превью — кнопки 5 последних использованных фото. Нажмите для быстрого повторного выбора.
📷
Требовать фото
Чекбокс 📷 Фото рядом с «Опубликовать». Когда включён — отправка без фото заблокирована. Защита от случайной публикации без изображения.

Задержка между отправками

Чекбокс Пауза в панели рассылки — включает случайную задержку между отправками. Задайте диапазон в полях от X до Y сек. Настройка сохраняется между запусками.

Помогает избежать спам-фильтров при массовой рассылке.

📤Отправка

Убедитесь что текст введён и нужные адреса отмечены галочками.
При необходимости загрузите фото (или снимите чекбокс 📷 Фото).
Нажмите Опубликовать. Кнопка покажет прогресс: «Отправка 2/5…»
По завершении появится лог с результатами и прозвучит сигнал.

Лог отправки

После рассылки в левой панели — список результатов:

  • ✓ Адрес — успешно отправлено
  • ✗ Адрес — ошибка при отправке

Кнопка 💾 Сохранить отчёт сохраняет лог в текстовый файл.

Отмена отправки

Во время рассылки отображается кнопка Отмена — прекращает отправку после текущего адреса.

Горячая клавиша публикации: Ctrl + Enter

🔀Умная рассылка

Умная рассылка — когда один текст содержит несколько блоков для разных адресов, разделённых пустыми строками. Каждый блок автоматически отправляется нужному получателю.

Пример текста
Уважаемые жители!

ул. Есенина д. 10
Плановое отключение воды 15 марта с 9:00 до 18:00.

ул. Есенина д. 14
Плановое отключение воды 16 марта с 9:00 до 18:00.

С уважением, УК «Пример»

Текст до первого блока — приветствие — добавляется в начало каждого.
Текст после последнего блока — подпись — добавляется в конец каждого.

Нажмите кнопку 🔀 Умная в заголовке «Ввод данных».
Откроется диалог предпросмотра — каждый блок с чекбоксом, видно кому и что пойдёт.
Нажмите Тест для пробного прогона (без реальной отправки), или Отправить.
Блоки разделяются пустой строкой. Каждый блок должен содержать адрес из реестра max_address.xlsx.

🕐Отложенная отправка

Планирование публикации в ВКонтакте на конкретное время.

Включите ☑ Отложить под кнопкой «Опубликовать».
Выберите дату и время публикации.
Нажмите Опубликовать — пост встанет в очередь ВКонтакте на указанное время.
Отложенная отправка работает только для ВКонтакте. Если программа была закрыта и пропустила время — при следующем запуске посты отправятся автоматически.

🔤Проверка орфографии

Поле ввода текста автоматически подчёркивает слова с возможными ошибками.

Нажмите правой кнопкой мыши на подчёркнутом слове.
В меню появятся варианты замены вида → правильное.
Нажмите нужный вариант — слово заменится автоматически.

Добавить в словарь

ПКМ на слове → «Добавить "слово" в словарь» — слово запомнится и больше не будет подчёркиваться.

Словарь хранится в %APPDATA%\MAX POST\user_dict.txt — можно редактировать вручную.

🔄Автообновления

MAX POST обновляется автоматически каждые 30 минут в фоне через планировщик задач Windows.

Фоновая задача проверяет наличие новой версии на сервере.
При наличии обновления — установщик скачивается и запускается автоматически.
Программа перезапускается. В трее: уведомление «Обновлено до vX.X.X».
Лог автообновления: %APPDATA%\MAX POST\update.log
Ручная проверка: меню Справка → Проверить обновления

История версий

Кнопка с иконкой обновления в боковой панели (над кнопкой авторизации) — показывает список изменений по версиям.

❓Частые вопросы

Программа не отправляет в MAX

Проверьте токен MAX в настройках → ✓ Проверить. Если истёк — получите новый по 🔗 Получить токен.

Адрес не находится в поиске

Убедитесь что адрес есть в max_address.xlsx. Обновите реестр: Действия → 🔄 Обновить реестр адресов.

Файл max_address.xlsx обновился — как перезагрузить?

Программа следит за изменением файла и покажет жёлтую полосу-уведомление. Нажмите «Обновить» или Действия → Обновить реестр адресов.

Не работает отправка в ВКонтакте

Токен ВК живёт около 3 месяцев. Проверьте его в настройках. После смены нажмите Действия → 🔑 Сменил токены VK.
